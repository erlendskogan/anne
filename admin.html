Sett opp Supabase – én gang

Følg disse stegene for å koble til databasen. Du gjør dette bare én gang.

  1. Opprett prosjekt
    Gå til supabase.com, logg inn og klikk «New project». Gi det et navn og velg en region nær Norge (f.eks. West EU (Ireland)).
  2. Opprett databasetabell
    Gå til SQL Editor i prosjektet og kjør dette: 
    create table nettside (
  id bigint primary key default 1,
  data jsonb not null default '{}'::jsonb,
  constraint kun_en_rad check (id = 1)
);

alter table nettside enable row level security;

create policy "Alle kan lese"
  on nettside for select using (true);

create policy "Innlogget kan skrive"
  on nettside for all to authenticated
  using (true) with check (true);
  3. Legg til brukerkontoer
    Gå til Authentication → Users → «Invite user». Opprett én konto per søster med e-post og passord.
  4. Kopier inn konfigurasjon
    Gå til Settings → API. Kopier «Project URL» og «anon public»-nøkkelen inn i filen supabase-config.js.

Når supabase-config.js er fylt inn, forsvinner denne guiden automatisk.
